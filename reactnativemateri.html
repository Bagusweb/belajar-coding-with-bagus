Materi React Native

1. Pengenalan React Native

React Native adalah framework yang memungkinkan kamu untuk membangun aplikasi mobile menggunakan JavaScript dan React. Dengan React Native, kamu dapat membuat aplikasi untuk iOS dan Android dengan basis kode yang sama.

2. Menginstal React Native

Untuk menginstal React Native, kamu perlu menginstal Node.js dan npm. Setelah itu, kamu dapat menggunakan Expo CLI atau React Native CLI untuk membuat proyek baru.

            npm install -g expo-cli
            expo init namaProyek

3. Struktur Proyek React Native

Struktur proyek React Native umumnya mencakup folder src, components, screens, dan assets. Berikut adalah contoh struktur dasar:

            namaProyek/
            ├── src/
            │   ├── components/
            │   ├── screens/
            │   └── assets/
            └── App.js

4. Komponen Dasar

Komponen di React Native adalah blok bangunan utama dari aplikasi. Berikut adalah contoh komponen dasar:

            import React from 'react';
            import { Text, View } from 'react-native';

            const HelloWorld = () => {
              return (
                
                  Hello, World!
                
              );
            };

            export default HelloWorld;

6. Navigasi

8. Menggunakan API

Kamu dapat menggunakan API untuk mengambil data dari server. Berikut adalah contoh penggunaan fetch untuk mengambil data:

            import React, { useEffect, useState } from 'react';
            import { Text, View } from 'react-native';

            const App = () => {
              const [data, setData] = useState(null);

              useEffect(() => {
                fetch('https://api.example.com/data')
                  .then(response => response.json())
                  .then(json => setData(json));
              }, []);

              return (
                
                  {data ? {data.title} : Loading...}
                
              );
            };

            export default App;

10. Testing Aplikasi

Untuk melakukan testing pada aplikasi React Native, kamu bisa menggunakan Jest atau React Native Testing Library. Berikut adalah contoh sederhana:

            import React from 'react';
            import { render } from '@testing-library/react-native';
            import App from './App';

            test('renders correctly', () => {
              const { getByText } = render();
              const linkElement = getByText(/Hello, World!/i);
              expect(linkElement).toBeTruthy();
            });

11. Publikasi Aplikasi

Setelah aplikasi selesai, kamu dapat mempublikasikannya ke App Store atau Google Play. Pastikan mengikuti panduan masing-masing platform untuk proses publikasi.

Kesimpulan

React Native adalah alat yang powerful untuk pengembangan aplikasi mobile. Dengan memahami konsep-konsep dasar yang telah dijelaskan, kamu dapat mulai membangun aplikasi mobile dengan lebih percaya diri.
