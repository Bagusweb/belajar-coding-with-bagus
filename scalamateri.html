Materi Scala

1. Pengertian Scala

Scala adalah bahasa pemrograman yang menggabungkan konsep dari pemrograman fungsional dan pemrograman berorientasi objek. Scala berjalan di atas Java Virtual Machine (JVM) dan dirancang untuk meningkatkan produktivitas pengembang.

2. Sejarah Scala

Scala pertama kali diperkenalkan oleh Martin Odersky pada tahun 2003. Tujuannya adalah untuk memperbaiki kekurangan Java dan menyediakan fitur pemrograman yang lebih modern.

3. Instalasi Scala

Untuk menginstal Scala, kamu perlu:

  • Instal JDK (Java Development Kit).
  • Unduh Scala dari situs resmi: scala-lang.org.
  • Ikuti petunjuk instalasi sesuai sistem operasi yang digunakan.

4. Hello World di Scala

Berikut adalah contoh program Hello World di Scala:

          object HelloWorld {
            def main(args: Array[String]): Unit = {
              println("Hello, World!")
            }
          }

5. Variabel dan Tipe Data

Scala mendukung berbagai tipe data seperti:

  • Int: Bilangan bulat.
  • Double: Bilangan desimal.
  • Boolean: Nilai true atau false.
  • String: Teks.

Contoh deklarasi variabel:

          val name: String = "John" // val untuk variabel konstan
          var age: Int = 25         // var untuk variabel yang dapat diubah

6. Fungsi di Scala

Fungsi di Scala didefinisikan menggunakan kata kunci def:

          def tambah(a: Int, b: Int): Int = {
            a + b
          }

Contoh pemanggilan fungsi:

          val hasil = tambah(5, 10)

7. Kontrol Alur

Scala mendukung struktur kontrol alur seperti if, for, dan while:

          if (age < 18) {
            println("Anak-anak")
          } else {
            println("Dewasa")
          }

          for (i <- 1 to 5) {
            println(i)
          }

8. Koleksi di Scala

Scala memiliki berbagai tipe koleksi, seperti List, Set, dan Map:

          val numbers = List(1, 2, 3, 4, 5)
          val uniqueNumbers = Set(1, 2, 2, 3)
          val ageMap = Map("John" -> 25, "Alice" -> 30)

9. Kelas dan Objek

Scala adalah bahasa berorientasi objek, yang berarti semuanya adalah objek. Berikut adalah contoh kelas dan objek:

          class Mobil(val merk: String, var tahun: Int) {
            def info(): String = s"$merk tahun $tahun"
          }

          val mobilSaya = new Mobil("Toyota", 2020)
          println(mobilSaya.info())

10. Penggunaan Trait

Trait adalah unit kode yang dapat digunakan kembali di dalam kelas. Berikut adalah contoh penggunaan trait:

          trait BisaBersuara {
            def suara(): String
          }

          class Kucing extends BisaBersuara {
            def suara(): String = "Meow"
          }

          val kucing = new Kucing
          println(kucing.suara())

11. Polimorfisme

Polimorfisme memungkinkan objek dari kelas yang berbeda untuk diakses melalui antarmuka yang sama. Berikut adalah contohnya:

          class Hewan {
            def suara(): String = "Suara Hewan"
          }

          class Anjing extends Hewan {
            override def suara(): String = "Guk"
          }

          def main(args: Array[String]): Unit = {
            val hewan: Hewan = new Anjing
            println(hewan.suara())
          }

12. Exception Handling

Scala memiliki mekanisme penanganan kesalahan menggunakan try, catch, dan finally:

          try {
            val result = 10 / 0
          } catch {
            case e: ArithmeticException => println("Tidak bisa membagi dengan nol")
          } finally {
            println("Blok finally akan dieksekusi")
          }

13. Kesimpulan

Scala adalah bahasa pemrograman yang powerful dengan sintaks yang bersih dan fitur yang canggih. Memahami konsep dasar Scala dapat meningkatkan kemampuan pengembangan perangkat lunak secara signifikan.
