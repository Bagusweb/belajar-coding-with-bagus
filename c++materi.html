Materi C++

1. Pengertian C++

C++ adalah bahasa pemrograman yang dikembangkan oleh Bjarne Stroustrup pada awal 1980-an. Bahasa ini merupakan pengembangan dari bahasa C dengan menambahkan konsep pemrograman berorientasi objek. C++ sangat populer untuk pengembangan perangkat lunak sistem, game, dan aplikasi yang membutuhkan performa tinggi.

2. Struktur Dasar Program C++

Program C++ dimulai dengan fungsi main() yang merupakan titik awal eksekusi program.

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

3. Tipe Data di C++

C++ memiliki berbagai tipe data yang dapat digunakan untuk menyimpan berbagai jenis nilai.

  • int: Bilangan bulat.
  • float: Bilangan desimal.
  • double: Bilangan desimal dengan presisi lebih tinggi.
  • char: Tipe data karakter.
  • string: Tipe data untuk menyimpan teks.

4. Struktur Kendali

C++ mendukung berbagai struktur kendali seperti perulangan dan percabangan.

  • IF Statement: Menggunakan if, else if, dan else untuk percabangan. 
    int umur = 20;
if (umur >= 18) {
    cout << "Dewasa" << endl;
} else {
    cout << "Belum Dewasa" << endl;
}
  • For Loop: Mengulang berdasarkan jumlah iterasi. 
    for (int i = 0; i < 5; i++) {
    cout << i << endl;
}
  • While Loop: Mengulang selama kondisi benar. 
    int angka = 1;
while (angka < 5) {
    cout << angka << endl;
    angka++;
}

5. Fungsi di C++

Fungsi digunakan untuk mengelompokkan blok kode yang dapat dipanggil berulang kali.

int tambah(int a, int b) {
    return a + b;
}

int main() {
    cout << tambah(3, 4) << endl;
}

6. Pemrograman Berorientasi Objek (OOP)

C++ mendukung pemrograman berorientasi objek (OOP) yang memungkinkan penggunaan kelas dan objek.

class Orang {
public:
    string nama;
    int umur;

    Orang(string n, int u) {
        nama = n;
        umur = u;
    }

    void perkenalan() {
        cout << "Nama saya " << nama << ", umur " << umur << endl;
    }
};

int main() {
    Orang john("John", 25);
    john.perkenalan();
}

7. Penanganan Error

C++ menggunakan mekanisme penanganan error dengan menggunakan try-catch.

try {
    int angka = 10;
    int hasil = angka / 0; // Pembagian dengan nol
} catch (const exception &e) {
    cout << "Terjadi error: " << e.what() << endl;
}

8. Pewarisan (Inheritance)

Pewarisan memungkinkan kelas untuk mewarisi sifat dan metode dari kelas lain.

class Hewan {
public:
    void suara() {
        cout << "Hewan bersuara." << endl;
    }
};

class Kucing : public Hewan {
public:
    void suara() {
        cout << "Kucing mengeong." << endl;
    }
};

int main() {
    Kucing kucing;
    kucing.suara(); // Output: Kucing mengeong.
}

9. Polimorfisme

Polimorfisme memungkinkan penggunaan metode yang sama dengan perilaku yang berbeda pada kelas yang berbeda.

class Bentuk {
public:
    virtual void gambar() {
        cout << "Gambar bentuk." << endl;
    }
};

class Lingkaran : public Bentuk {
public:
    void gambar() override {
        cout << "Gambar lingkaran." << endl;
    }
};

int main() {
    Bentuk* b;
    Lingkaran l;
    b = &l;
    b->gambar(); // Output: Gambar lingkaran.
}

10. Template

Template memungkinkan pembuatan fungsi dan kelas yang dapat bekerja dengan tipe data yang berbeda.

template <typename T>
T tambah(T a, T b) {
    return a + b;
}

int main() {
    cout << tambah(3, 4) << endl; // Output: 7
    cout << tambah(3.5, 2.5) << endl; // Output: 6
}

11. Standar Template Library (STL)

STL menyediakan kumpulan kelas template dan fungsi untuk manipulasi data.

#include <vector>

int main() {
    vector<int> v = {1, 2, 3, 4, 5};
    for (int i : v) {
        cout << i << " ";
    }
    // Output: 1 2 3 4 5
}

12. Operator Overloading

Operator overloading memungkinkan kita untuk mendefinisikan perilaku operator untuk kelas kustom.

class Bilangan {
public:
    int nilai;
    Bilangan(int n) : nilai(n) {}
    
    Bilangan operator+(const Bilangan &b) {
        return Bilangan(nilai + b.nilai);
    }
};

int main() {
    Bilangan a(10), b(20);
    Bilangan c = a + b;
    cout << c.nilai << endl; // Output: 30
}

13. Namespace

Namespace digunakan untuk mengelompokkan kode dan mencegah konflik nama.

namespace Matematika {
    int tambah(int a, int b) {
        return a + b;
    }
}

int main() {
    cout << Matematika::tambah(5, 10) << endl; // Output: 15
}

14. Fungsi Lambda

Fungsi lambda adalah fungsi anonim yang dapat didefinisikan dalam ekspresi.

auto tambah = [](int a, int b) {
    return a + b;
};

int main() {
    cout << tambah(3, 4) << endl; // Output: 7
}

15. Smart Pointers

Smart pointers mengelola memori secara otomatis dan mencegah kebocoran memori.

#include <memory>

int main() {
    unique_ptr<int> ptr = make_unique<int>(5);
    cout << *ptr << endl; // Output: 5
}

Kesimpulan

C++ adalah bahasa pemrograman yang kuat dan fleksibel, mendukung berbagai paradigma pemrograman, termasuk pemrograman prosedural dan berorientasi objek. Dengan fitur seperti tipe data yang kaya, struktur kendali, dan pemrograman berorientasi objek, C++ sering digunakan dalam pengembangan perangkat lunak yang memerlukan performa tinggi. Meskipun mungkin memiliki kurva belajar yang lebih tinggi dibandingkan bahasa pemrograman lain, penguasaan C++ dapat membuka banyak peluang dalam dunia teknologi.
