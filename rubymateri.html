Materi Ruby

1. Apa itu Ruby dan Sejarahnya

Ruby adalah bahasa pemrograman yang dinamis, reflektif, dan berorientasi objek. Dikembangkan oleh Yukihiro Matsumoto pada tahun 1995, Ruby dirancang untuk kemudahan penggunaan dan produktivitas. Sejak pertama kali dirilis, bahasa ini telah berkembang pesat dengan rilis versi terbaru yang terus dilakukan untuk menambah fitur dan meningkatkan performa.

2. Fitur Utama Ruby

  • Bahasa berorientasi objek.
  • Sintaks yang mudah dibaca dan ditulis.
  • Dukungan untuk pengembangan aplikasi web melalui framework Ruby on Rails.

3. Instalasi Ruby

Untuk menginstal Ruby, Anda bisa menggunakan rbenv atau rvm. Berikut adalah contoh instalasi menggunakan rbenv:

$ brew install rbenv
$ rbenv install 3.0.0
$ rbenv global 3.0.0

4. Sintaks Dasar Ruby

Berikut adalah contoh sintaks dasar dalam Ruby:

puts "Hello, World!"

5. Variabel dan Tipe Data

Ruby mendukung berbagai tipe data, termasuk:

  • Integer: Angka bulat.
  • Float: Angka desimal.
  • String: Teks.

Contoh mendeklarasikan variabel:

age = 25
name = "Alice"

6. Kontrol Alur

Ruby menggunakan struktur kontrol seperti if, else, dan case. Contoh penggunaan if:

if age >= 18
  puts "Dewasa"
else
  puts "Belum dewasa"
end

7. Fungsi dalam Ruby

Fungsi dalam Ruby dideklarasikan dengan kata kunci def. Contoh fungsi sederhana:

def greet(name)
  puts "Hello, #{name}!"
end

greet("Alice")

8. Array dan Hash

Array adalah koleksi terurut, sedangkan Hash adalah koleksi pasangan kunci-nilai.

fruits = ["apple", "banana", "orange"]
person = { name: "Alice", age: 25 }

9. Kesalahan dan Penanganannya

Ruby memiliki mekanisme penanganan kesalahan menggunakan begin, rescue, dan ensure.

begin
  # Kode yang mungkin menyebabkan kesalahan
rescue StandardError => e
  puts "Terjadi kesalahan: #{e.message}"
ensure
  puts "Ini akan selalu dijalankan."
end

Kesimpulan

Ruby adalah bahasa pemrograman yang powerful dan mudah digunakan, terutama untuk pengembangan web dengan Ruby on Rails. Dengan sintaks yang sederhana, Ruby cocok untuk pemula maupun pengembang berpengalaman.
