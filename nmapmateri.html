Materi Nmap

1. Pengenalan Nmap

Nmap (Network Mapper) adalah alat open-source untuk pemindai jaringan dan audit keamanan. Nmap digunakan untuk menemukan perangkat yang terhubung ke jaringan dan menilai keamanannya.

2. Instalasi Nmap

Untuk menginstal Nmap, gunakan perintah berikut sesuai distribusi Linux kamu:

  • Debian/Ubuntu: sudo apt install nmap
  • CentOS: sudo yum install nmap

3. Memindai Host

Untuk memindai sebuah host, gunakan perintah berikut:

            nmap [alamat_IP]

Contoh:

            nmap 192.168.1.1

4. Memindai Port

Untuk memindai port tertentu pada host, gunakan perintah:

            nmap -p [port] [alamat_IP]

Contoh untuk memindai port 22 dan 80:

            nmap -p 22,80 192.168.1.1

5. Memindai Seluruh Port

Untuk memindai semua port pada host, gunakan opsi:

            nmap -p- [alamat_IP]

6. Memindai dengan Deteksi OS

Untuk mendeteksi sistem operasi target, gunakan opsi:

            nmap -O [alamat_IP]

7. Memindai dengan Skrip

Nmap memiliki fitur skrip yang memungkinkan pemindaian lanjutan. Untuk menjalankan skrip, gunakan:

            nmap --script [nama_skrip] [alamat_IP]

8. Pemindaian Tercepat

Untuk mempercepat pemindaian, gunakan opsi:

            nmap -T4 [alamat_IP]

9. Menyimpan Hasil Pemindaian

Untuk menyimpan hasil pemindaian ke dalam file, gunakan opsi:

            nmap -oN hasil.txt [alamat_IP]

10. Pemindaian dalam Mode Stealth

Untuk melakukan pemindaian stealth, gunakan opsi:

            nmap -sS [alamat_IP]

11. Pemindaian UDP

Untuk melakukan pemindaian pada port UDP, gunakan opsi:

            nmap -sU [alamat_IP]

12. Kesimpulan

Nmap adalah alat yang sangat berguna untuk keamanan jaringan dan pemetaan. Memahami berbagai opsi dan cara penggunaannya sangat penting untuk melakukan audit jaringan yang efektif.
