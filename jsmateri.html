Materi JavaScript

1. Sejarah dan Pengertian JavaScript

Pengertian JavaScript: JavaScript adalah bahasa pemrograman tingkat tinggi yang awalnya dikembangkan untuk membuat halaman web menjadi interaktif. Sekarang, JavaScript digunakan tidak hanya di sisi klien (browser) tetapi juga di sisi server (Node.js).

Sejarah JavaScript: JavaScript pertama kali dikembangkan oleh Brendan Eich pada tahun 1995 saat bekerja di Netscape. Pada awalnya dikenal sebagai Mocha, lalu berganti nama menjadi LiveScript, dan akhirnya menjadi JavaScript. JavaScript distandarisasi sebagai ECMAScript oleh ECMA International.

2. Dasar-Dasar JavaScript

a. Struktur Dasar JavaScript

  • Variabel: Digunakan untuk menyimpan data. 
    var x = 5;
let y = 10;
const z = 15;
  • Tipe Data: JavaScript mendukung berbagai tipe data seperti string, number, boolean, array, object, dan lain-lain. 
    let nama = "John"; // String
let umur = 25; // Number
let isStudent = true; // Boolean
let hobbies = ["membaca", "menulis"]; // Array
let person = { nama: "John", umur: 25 }; // Object
  • Operator: JavaScript mendukung operator aritmatika (+, -, *, /), perbandingan (==, ===, !=, !==), dan logika (&&, ||, !).

b. Fungsi (Function)

  • Definisi Fungsi: Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu dan dapat digunakan kembali. 
    function greet(name) {
  return "Hello, " + name;
}
  • Fungsi Arrow (Arrow Function): Sintaks singkat untuk membuat fungsi. 
    const greet = (name) => "Hello, " + name;

c. Kontrol Alur (Control Flow)

  • Percabangan (if, else if, else) 
    let umur = 18;
if (umur >= 18) {
  console.log("Dewasa");
} else {
  console.log("Belum Dewasa");
}
  • Looping: Perulangan digunakan untuk menjalankan blok kode beberapa kali. 
    // For loop
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While loop
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

3. Manipulasi DOM (Document Object Model)

JavaScript digunakan untuk mengakses dan memanipulasi elemen HTML melalui DOM.

  • Mengakses Elemen: 
    let element = document.getElementById("myElement");
let elements = document.getElementsByClassName("myClass");
let queryElement = document.querySelector(".myClass");
  • Mengubah Konten: 
    element.innerHTML = "Hello, World!";
element.style.color = "red";

4. Event Handling

Event adalah aksi yang dilakukan oleh pengguna atau browser, seperti klik, scroll, dan input.

  • Mendengarkan Event: 
    document.getElementById("myButton").addEventListener("click", function() {
  alert("Button clicked!");
});

5. JavaScript Asinkron

a. Promises

Promise digunakan untuk menangani operasi asinkron, seperti memuat data dari server.

let promise = new Promise((resolve, reject) => {
  // Kode asinkron
  let success = true;
  if (success) {
    resolve("Berhasil");
  } else {
    reject("Gagal");
  }
});

promise.then(result => {
  console.log(result);
}).catch(error => {
  console.log(error);
});

b. Async/Await

Async/await adalah cara modern untuk menangani kode asinkron yang membuat kode lebih mudah dibaca.

async function fetchData() {
  try {
    let response = await fetch("https://api.example.com/data");
    let data = await response.json();
    console.log(data);
  } catch (error) {
    console.log(error);
  }
}

fetchData();

6. Pemrograman Berorientasi Objek (OOP)

JavaScript mendukung OOP yang memungkinkan kita membuat objek dan menggunakan prinsip pewarisan.

a. Class dan Object

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    return "Hello, " + this.name;
  }
}

let john = new Person("John", 25);
console.log(john.greet());

7. Teknik dan Fitur Lanjutan JavaScript

  • Closures: Fungsi yang mengingat lingkup variabel saat ia didefinisikan. 
    function outer() {
  let count = 0;
  function inner() {
    count++;
    return count;
  }
  return inner;
}

let counter = outer();
console.log(counter()); // Output: 1
console.log(counter()); // Output: 2
  • Modules: Membuat dan mengimpor modul untuk memisahkan kode. 
    // Module file: math.js
export function add(x, y) {
  return x + y;
}

// Main file
import { add } from './math.js';
console.log(add(2, 3));

Kesimpulan

JavaScript: JavaScript adalah bahasa pemrograman yang menambahkan interaktivitas dan fungsionalitas dinamis ke halaman web. Dengan JavaScript, kita dapat membuat elemen yang dapat diubah sesuai input pengguna, menjalankan proses otomatis di halaman, dan berinteraksi dengan server untuk memberikan pengalaman yang lebih kaya.
