Materi PHP

1. Pengertian PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman yang dirancang untuk pengembangan web. PHP berjalan di sisi server, artinya kode PHP dieksekusi di server dan hasilnya dikirim ke browser pengguna. PHP sangat cocok untuk membuat aplikasi web dinamis, di mana konten halaman dapat berubah sesuai dengan permintaan pengguna.

Beberapa fitur utama PHP antara lain:

  • Memungkinkan interaksi dengan database.
  • Mendukung berbagai jenis database, termasuk MySQL dan PostgreSQL.
  • Dapat digunakan untuk mengelola sesi pengguna dan cookie.
  • Memiliki banyak pustaka dan framework yang memudahkan pengembangan aplikasi.

2. Struktur Dasar PHP

Untuk memulai menulis kode PHP, kamu perlu menyimpan kode tersebut dalam file dengan ekstensi .php. Kode PHP dibuka dan ditutup dengan tag <?php dan ?>.

<?php echo "Halo, dunia!"; ?>

Dalam contoh di atas, echo digunakan untuk mencetak string ke layar.

3. Variabel dalam PHP

Variabel dalam PHP digunakan untuk menyimpan data. Setiap variabel diawali dengan tanda dolar ($). PHP tidak memerlukan deklarasi tipe data secara eksplisit; tipe data ditentukan otomatis oleh nilai yang diberikan.

<?php 
$nama = "John"; 
$umur = 25; 
echo "Nama saya adalah $nama dan saya berumur $umur tahun."; 
?>

Variabel juga bisa menyimpan berbagai tipe data seperti integer, string, array, dan objek.

4. Pernyataan Kondisi (If-Else)

Pernyataan kondisi digunakan untuk menentukan alur program berdasarkan kondisi tertentu. Berikut adalah contoh penggunaannya:

<?php 
$umur = 18; 
if ($umur >= 18) { 
    echo "Anda sudah dewasa."; 
} else { 
    echo "Anda belum cukup umur."; 
} 
?>

Dalam contoh di atas, jika variabel $umur lebih besar atau sama dengan 18, maka program akan mencetak "Anda sudah dewasa." Jika tidak, akan mencetak "Anda belum cukup umur."

5. Looping (Perulangan) dalam PHP

PHP mendukung beberapa jenis looping, seperti for, while, dan foreach. Looping memungkinkan kita untuk menjalankan blok kode berulang kali.

Contoh penggunaan loop for:

<?php 
for ($i = 0; $i < 5; $i++) { 
    echo "Perulangan ke-$i<br>"; 
} 
?>

Di sini, loop for digunakan untuk mencetak angka dari 0 hingga 4.

6. Fungsi dalam PHP

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu. Fungsi dapat menerima parameter dan mengembalikan nilai. Berikut adalah contoh cara mendeklarasikan dan menggunakan fungsi:

<?php 
function sapaPengguna($nama) { 
    return "Halo, $nama!"; 
} 
echo sapaPengguna("John"); 
?>

Fungsi sapaPengguna menerima satu parameter, $nama, dan mengembalikan string sapaan yang disesuaikan.

7. Menggunakan PHP dengan Formulir HTML

Kamu bisa mengumpulkan data dari pengguna menggunakan formulir HTML dan memprosesnya dengan PHP. Berikut adalah contoh formulir sederhana:

<form action="proses.php" method="POST"> 
    <input type="text" name="nama"> 
    <input type="submit" value="Kirim"> 
</form>

Data yang dimasukkan ke dalam formulir akan dikirim ke file proses.php untuk diproses lebih lanjut.

8. Koneksi Database dengan PHP

PHP dapat terhubung dengan berbagai database, termasuk MySQL. Berikut adalah contoh cara membuat koneksi ke database MySQL:

<?php 
$servername = "localhost"; 
$username = "root"; 
$password = ""; 
$dbname = "nama_database"; 

// Membuat koneksi
$conn = new mysqli($servername, $username, $password, $dbname);

// Memeriksa koneksi
if ($conn->connect_error) { 
    die("Koneksi gagal: " . $conn->connect_error); 
} 
echo "Koneksi berhasil!"; 
?>

Pastikan untuk mengganti nama_database dengan nama database yang sesuai. Kode ini akan menghubungkan PHP dengan database dan memberi tahu jika koneksi berhasil.

9. Keamanan dalam PHP

Keamanan adalah aspek penting dalam pengembangan aplikasi web. Beberapa teknik untuk melindungi aplikasi PHP meliputi:

  • Sanitasi dan validasi input pengguna untuk mencegah serangan SQL Injection.
  • Gunakan prepared statements saat berinteraksi dengan database.
  • Enkripsi data sensitif, seperti password.
  • Gunakan HTTPS untuk komunikasi aman antara server dan klien.

Implementasi keamanan yang baik dapat mencegah berbagai serangan dan menjaga data pengguna tetap aman.

10. Contoh Kode PHP

Berikut adalah beberapa contoh tambahan untuk memperdalam pemahaman kamu tentang PHP:

  1. Menampilkan tanggal dan waktu saat ini: 
    <?php 
echo date("Y-m-d H:i:s"); 
?>
  2. Membuat array dan menampilkannya: 
    <?php 
$buah = array("Apel", "Pisang", "Jeruk"); 
foreach ($buah as $item) { 
    echo $item . "<br>"; 
} 
?>
  3. Menangani kesalahan dengan try-catch: 
    <?php 
try { 
    // Kode yang mungkin menghasilkan kesalahan 
    throw new Exception("Terjadi kesalahan!"); 
} catch (Exception $e) { 
    echo "Pesan kesalahan: " . $e->getMessage(); 
} 
?>
  4. Upload file dengan PHP: 
    <form action="upload.php" method="post" enctype="multipart/form-data"> 
    Pilih file untuk diunggah: 
    <input type="file" name="fileToUpload"> 
    <input type="submit" value="Unggah"> 
</form>
  5. Menggunakan session untuk menyimpan data pengguna: 
    <?php 
session_start(); 
$_SESSION["username"] = "JohnDoe"; 
echo "Username disimpan di session: " . $_SESSION["username"]; 
?>
  6. Membuat class dan objek: 
    <?php 
class Mobil { 
    public $warna; 
    public function __construct($warna) { 
        $this->warna = $warna; 
    } 
} 
$mobilBaru = new Mobil("Merah"); 
echo "Warna mobil baru: " . $mobilBaru->warna; 
?>
  7. Menangani form dengan metode GET: 
    <form action="proses.php" method="GET"> 
    Nama: <input type="text" name="nama"> 
    <input type="submit" value="Kirim"> 
</form>
  8. Menggunakan regex untuk validasi: 
    <?php 
$email = "user@example.com"; 
if (filter_var($email, FILTER_VALIDATE_EMAIL)) { 
    echo "Email valid."; 
} else { 
    echo "Email tidak valid."; 
} 
?>
  9. Pengulangan menggunakan while: 
    <?php 
$angka = 0; 
while ($angka < 5) { 
    echo "Angka: $angka<br>"; 
    $angka++; 
} 
?>
  10. Menghitung jumlah karakter dalam string: 
    <?php 
$string = "Halo Dunia!"; 
echo "Jumlah karakter: " . strlen($string); 
?>

Kesimpulan

PHP adalah bahasa pemrograman yang kuat untuk pengembangan web. Dengan memahami dasar-dasar PHP, kamu dapat membuat aplikasi web dinamis dan interaktif. Penting untuk terus belajar dan mengasah keterampilan kamu dalam PHP agar dapat membuat aplikasi yang lebih kompleks dan aman.
