Materi HTML

1. Sejarah dan Pengertian HTML

Pengertian HTML:

Apa itu HTML? HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. HTML mendefinisikan struktur dan layout halaman web dengan menggunakan elemen-elemen seperti heading, paragraf, link, gambar, dan lainnya. HTML bukanlah bahasa pemrograman, melainkan bahasa markup yang memberikan instruksi kepada browser bagaimana menampilkan konten.

Sejarah HTML:

  • Diciptakan oleh Tim Berners-Lee (1991): HTML dikembangkan pertama kali oleh Tim Berners-Lee, seorang ilmuwan komputer asal Inggris. Pada tahun 1991, ia merilis versi pertama HTML sebagai alat untuk berbagi informasi dalam jaringan World Wide Web (WWW) yang juga ia ciptakan. Versi pertama HTML ini sangat sederhana dan hanya mencakup 18 tag dasar yang membantu penyusunan teks dan struktur halaman.
  • HTML 2.0 (1995): Setelah HTML mulai banyak digunakan, HTML 2.0 diperkenalkan pada tahun 1995 oleh Internet Engineering Task Force (IETF). Versi ini menambahkan fitur baru seperti form (untuk menangani input pengguna) dan tag yang lebih canggih. HTML 2.0 menjadi versi resmi pertama yang distandarisasi.
  • HTML 3.2 dan 4.0 (1997 dan 1999): HTML 3.2 dirilis pada tahun 1997 oleh World Wide Web Consortium (W3C), organisasi yang bertanggung jawab atas standarisasi HTML. Versi ini memperkenalkan elemen tata letak yang lebih kompleks seperti tabel dan dukungan untuk script. HTML 4.0 (1999) membawa pembaruan besar dengan memisahkan konten dan presentasi menggunakan CSS (Cascading Style Sheets).
  • XHTML (2000): XHTML adalah evolusi dari HTML 4.0 yang menambahkan prinsip-prinsip XML untuk membuat kode HTML lebih ketat dan konsisten. Namun, karena persyaratan sintaks yang ketat, XHTML tidak menjadi populer di kalangan pengembang.
  • HTML5 (2014): HTML5 dirilis oleh W3C dan WHATWG sebagai versi terbaru dan paling canggih dari HTML. HTML5 memperkenalkan banyak fitur baru, seperti elemen multimedia (`<audio>`, `<video>`, `<canvas>`), tag baru untuk struktur semantik (`<header>`, `<footer>`, `<article>`, `<section>`, dll.), serta mendukung aplikasi web interaktif dan aksesibilitas yang lebih baik.

HTML terus berkembang, dengan pembaruan kecil yang ditambahkan oleh WHATWG, yang sekarang menjaga HTML sebagai standar yang terus berkembang untuk memenuhi kebutuhan aplikasi web modern.

2. Struktur Dasar HTML

Setiap halaman HTML memiliki struktur dasar yang sama, yang terdiri dari elemen-elemen berikut:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
  </head>
  <body>
    <!-- Konten halaman web -->
  </body>
</html>
  • <!DOCTYPE html>: Deklarasi bahwa dokumen ini menggunakan HTML5.
  • <html>: Elemen root yang mencakup seluruh konten.
  • <head>: Bagian metadata (seperti judul, link ke stylesheet, charset).
  • <body>: Tempat konten halaman web ditampilkan.

3. Elemen Dasar HTML

a. Heading

Heading digunakan untuk membuat judul dalam berbagai ukuran, dari <h1> (terbesar) hingga <h6> (terkecil).

<h1>Ini Heading 1</h1>
<h2>Ini Heading 2</h2>
...
<h6>Ini Heading 6</h6>


b. Paragraf

Elemen <p> digunakan untuk membuat blok teks sebagai paragraf.

<p>Ini adalah sebuah paragraf.</p>


c. Format Teks

  • Bold: 
    <strong>Ini teks tebal</strong>
  • Italic: 
    <em>Ini teks miring</em>
  • Underline: 
    <u>Ini teks bergaris bawah</u>


d. Link

Elemen <a> digunakan untuk membuat hyperlink.

<a href="https://www.example.com">Ini adalah link</a>


e. Gambar

Menggunakan elemen <img> untuk menampilkan gambar.

<img src="gambar.jpg" alt="Deskripsi gambar">
  • src: Menunjukkan lokasi gambar.
  • alt: Teks alternatif jika gambar tidak dapat dimuat.


f. Daftar

Ordered List (<ol>) untuk daftar bernomor.

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Unordered List (<ul>) untuk daftar dengan bullet.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

4. Form HTML

HTML menyediakan berbagai elemen untuk membuat form interaktif. Form digunakan untuk mengumpulkan data dari pengguna dan mengirimkannya ke server.

<form action="/submit" method="post">
  <label for="name">Nama:</label>
  <input type="text" id="name" name="name">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Submit">
</form>
  • <form>: Elemen utama untuk <form>.
  • action: URL tempat data dikirim.
  • method: URL Metode pengiriman data (get atau post).
  • <input type="text">: Elemen input untuk menerima data pengguna

5. Tabel HTML

Tabel digunakan untuk menyajikan data dalam format baris dan kolom. Struktur tabel HTML terdiri dari elemen <table>, <tr> (baris), <th> (header kolom), dan <td> (data sel).

<table>
  <tr>
    <th>Nama</th>
    <th>Umur</th>
  </tr>
  <tr>
    <td>Jhon</td>
    <td>20</td>
  </tr>
</table>

6. Fitur Lanjutan HTML

a. Multimedia (Audio & Video)

HTML5 memperkenalkan elemen <audio> dan <video> untuk menambahkan media tanpa plugin eksternal.

  • Audio 
    <audio controls>
  <source src="audio.mp3" type="audio/mp3">
</audio>

  • Video 
    <video controls>
  <source src="video.mp4" type="video/mp4">
</video>


b. Canvas (Gambar Dinamis)

Elemen <canvas> digunakan untuk menggambar grafik menggunakan JavaScript.

<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  let c = document.getElementById("myCanvas");
  let ctx = c.getContext("2d");
  ctx.fillStyle = "#FF0000";
  ctx.fillRect(20, 20, 150, 100);
</script>


c. SVG (Scalable Vector Graphics)

HTML5 juga mendukung grafik vektor menggunakan elemen <svg>.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>


d. Media Queries (Responsif)

Media queries digunakan untuk membuat halaman responsif berdasarkan ukuran layar perangkat.

<style>
  body {
    background-color: lightblue;
  }

  @media screen and (max-width: 600px) {
    body {
      background-color: lightgreen;
    }
  }
</style>

7. Integrasi CSS dengan HTML

a. Internal CSS

CSS bisa ditulis langsung dalam file HTML di dalam elemen <style>.

<style>
  body {
    background-color: #f0f0f0;
  }
</style>


b. External CSS

Lebih baik memisahkan CSS ke file eksternal.

<link rel="stylesheet" href="styles.css">

8. Integrasi JavaScript dengan HTML

JavaScript ditambahkan ke HTML untuk interaktivitas, baik secara internal maupun melalui file eksternal.

<script>
  document.getElementById("demo").innerHTML = "Hello World!";
</script>

9. SEO dan Aksesibilitas

a. Meta Tags

Meta tags seperti <meta name="description" content="Belajar HTML"> digunakan untuk membantu SEO.


b. Aksesibilitas

Gunakan elemen seperti <alt> untuk gambar, role untuk elemen interaktif, dan struktur heading yang logis untuk meningkatkan aksesibilitas.

10. HTML5 API dan Fitur Lanjutan

a. Geolocation API

Mengambil lokasi geografis pengguna.

<script>
  navigator.geolocation.getCurrentPosition(function(position) {
    console.log(position.coords.latitude, position.coords.longitude);
  });
</script>


b. Local Storage

HTML5 memungkinkan penyimpanan data di browser dengan localStorage.

<script>
  localStorage.setItem("name", "John");
  alert(localStorage.getItem("name"));
</script>

Kesimpulan

HTML (HyperText Markup Language): HTML adalah bahasa markup yang berfungsi membangun struktur dasar halaman web. Elemen-elemen HTML seperti tag heading, paragraf, gambar, dan link menyusun konten halaman, memberikan hierarki informasi, dan menyiapkan fondasi bagi desain visual.
