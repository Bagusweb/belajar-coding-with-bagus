Materi CSS

1. Sejarah dan Pengertian CSS

Pengertian CSS: CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan dan format halaman web yang ditulis dalam HTML. Dengan CSS, kita bisa mengatur warna, font, tata letak, dan gaya visual lainnya.

Sejarah CSS: CSS pertama kali dikembangkan pada tahun 1996 oleh World Wide Web Consortium (W3C). Versi pertama (CSS1) hanya mendukung format dasar. CSS2 (1998) dan CSS3 (2001) memperkenalkan fitur yang lebih kompleks seperti tata letak grid, efek animasi, dan lainnya.

2. Dasar-Dasar CSS

a. Struktur CSS

  • Selektor (Selector): Menentukan elemen HTML mana yang akan di-styling.
  • Deklarasi (Declaration): Terdiri dari properti (property) dan nilai (value), misalnya color: red;.
  • Aturan CSS (CSS Rule): Kombinasi selektor dan deklarasi.
/* Contoh aturan CSS */
p {
  color: blue;
  font-size: 16px;
}

b. Cara Mengintegrasikan CSS dengan HTML

  • CSS Inline: Menulis CSS langsung di elemen HTML. 
    <p style="color: red;">Teks berwarna merah.</p>
  • CSS Internal: Menulis CSS di dalam tag <style> pada bagian <head>. 
    <style>
  p { color: green; }
</style>
  • CSS Eksternal: Menyimpan CSS dalam file .css terpisah dan menghubungkannya di HTML. 
    <link rel="stylesheet" href="style.css">

c. Selektor CSS

  • Selektor Tag: Memilih semua elemen HTML dengan nama tag tertentu. 
    h1 { color: blue; }
  • Selektor ID: Memilih elemen dengan atribut id tertentu. 
    #header { background-color: grey; }
  • Selektor Class: Memilih elemen dengan atribut class tertentu. 
    .content { font-size: 14px; }
  • Selektor Atribut: Memilih elemen berdasarkan atribut tertentu. 
    input[type="text"] { color: green; }

d. Properti Dasar CSS

  • Warna (Color): Mengatur warna teks, latar belakang, border. 
    color: red;  /* Warna teks */
background-color: yellow;  /* Warna latar belakang */
  • Font dan Teks: 
    font-family: Arial, sans-serif;
font-size: 16px;
font-weight: bold;
text-align: center;
  • Margin dan Padding: Mengatur jarak di luar (margin) dan di dalam (padding) elemen. 
    margin: 20px;
padding: 10px;
  • Border: Mengatur garis tepi elemen. 
    border: 1px solid black;

3. Konsep Tata Letak (Layout) dalam CSS

a. Display

  • Display Block dan Inline: Mengatur cara elemen ditampilkan. 
    display: block;   /* Mengisi ruang secara penuh */
display: inline;  /* Mengisi ruang sesuai konten */
  • Display Flexbox: Digunakan untuk membuat tata letak fleksibel. 
    display: flex;
justify-content: center;
align-items: center;
  • Display Grid: Mengatur layout berbasis grid yang dinamis. 
    display: grid;
grid-template-columns: 1fr 1fr;
grid-gap: 10px;

b. Positioning

  • Position Static: Default positioning. 
    position: static;
  • Position Relative: Menempatkan elemen relatif terhadap posisi aslinya. 
    position: relative;
top: 10px;  /* Bergeser 10px dari posisi asli */
  • Position Absolute: Menempatkan elemen relatif terhadap elemen terdekat yang diposisikan. 
    position: absolute;
top: 0; right: 0;
  • Position Fixed: Memperbaiki elemen pada posisi tertentu saat scroll. 
    position: fixed;
top: 0; left: 0;

4. Media Queries

Pengertian: Media queries digunakan untuk menerapkan gaya yang berbeda untuk perangkat yang berbeda berdasarkan karakteristik seperti lebar layar.

/* Contoh media query */
@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

Dengan media queries, kita dapat membuat desain responsif yang menyesuaikan dengan berbagai ukuran layar.

5. Animasi dan Transisi CSS

a. Transisi

Transisi CSS memungkinkan perubahan gradual dari satu gaya ke gaya lainnya.

/* Contoh transisi */
.button {
  transition: background-color 0.5s ease;
}

.button:hover {
  background-color: blue;
}

b. Animasi

Animasi CSS memungkinkan untuk membuat efek gerakan yang lebih kompleks.

/* Contoh animasi */
@keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}

.box {
  animation: example 5s infinite;
}

6. Flexbox

Flexbox adalah model layout satu dimensi yang membantu dalam distribusi ruang dan penyelarasan elemen dalam wadah (container).

.container {
  display: flex;
  justify-content: space-between; /* Mengatur jarak antar elemen */
}
.item {
  flex: 1; /* Mengizinkan item untuk tumbuh sama */
}

7. CSS Grid

CSS Grid adalah model layout dua dimensi yang memungkinkan untuk mengatur elemen dalam baris dan kolom.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* Tiga kolom sama lebar */
  grid-gap: 10px; /* Jarak antar elemen */
}
.grid-item {
  background-color: lightgrey;
}

8. CSS Variables

CSS Variables (custom properties) memungkinkan kita untuk menyimpan nilai yang dapat digunakan kembali dalam stylesheet.

:root {
  --main-color: #3498db; /* Variabel warna utama */
}

body {
  color: var(--main-color); /* Menggunakan variabel */
}

9. CSS Frameworks

CSS Frameworks adalah kumpulan file CSS yang sudah ditulis sebelumnya yang memudahkan pengembangan.

  • Bootstrap: Framework populer untuk desain responsif.
  • Tailwind CSS: Utility-first CSS framework yang memungkinkan kustomisasi lebih.
  • Bulma: Framework CSS modern yang mudah digunakan dan responsif.

10. CSS Preprocessors

CSS Preprocessors adalah alat yang memungkinkan kita untuk menulis CSS lebih efisien dengan fitur seperti variabel, nested rules, dan fungsi.

  • Sass: Preprocessor yang populer dengan sintaks yang kuat.
  • Less: Preprocessor dengan sintaks mirip CSS yang mudah digunakan.
  • Stylus: Preprocessor fleksibel yang mendukung sintaks tanpa kurung atau titik koma.

Kesimpulan

CSS adalah alat yang sangat penting dalam pengembangan web, karena memberikan kontrol penuh atas tampilan dan nuansa situs. Dengan memahami dasar-dasar CSS, serta konsep-konsep seperti Flexbox, Grid, dan penggunaan media queries, kita dapat menciptakan desain yang responsif dan menarik. Penguasaan CSS juga membuka jalan untuk eksplorasi lebih lanjut ke dalam CSS Frameworks dan Preprocessors yang meningkatkan efisiensi dan kemampuan styling.
