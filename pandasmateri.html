Materi Pandas

1. Pengertian Pandas

Pandas adalah pustaka open-source untuk bahasa pemrograman Python yang digunakan untuk analisis data dan manipulasi data. Pandas menyediakan struktur data yang mudah digunakan, seperti DataFrame dan Series.

2. Instalasi Pandas

Untuk menginstal Pandas, kamu dapat menggunakan pip. Berikut adalah perintahnya:

                        pip install pandas

3. Struktur Data Pandas

Pandas memiliki dua struktur data utama:

  • Series: Struktur data satu dimensi yang mirip dengan array atau list.
  • DataFrame: Struktur data dua dimensi yang mirip dengan tabel SQL atau spreadsheet.

4. Membuat DataFrame

Berikut adalah contoh cara membuat DataFrame:

                        import pandas as pd

                        data = {
                            'Nama': ['Alice', 'Bob', 'Charlie'],
                            'Umur': [25, 30, 35]
                        }
                        df = pd.DataFrame(data)

5. Menampilkan DataFrame

Untuk menampilkan DataFrame, cukup panggil nama DataFrame:

                        print(df)

6. Mengakses Kolom dan Baris

Kamu bisa mengakses kolom dengan nama kolom:

                        df['Nama']

Dan mengakses baris menggunakan iloc untuk indeks:

                        df.iloc[0]  # Mengakses baris pertama

7. Menambah Kolom

Untuk menambah kolom baru ke DataFrame:

                        df['Kota'] = ['Jakarta', 'Bandung', 'Semarang']

8. Menghapus Kolom

Untuk menghapus kolom dari DataFrame:

                        df.drop('Kota', axis=1, inplace=True)

9. Menyaring Data

Kamu dapat menyaring data dengan kondisi:

                        df[df['Umur'] > 30]

10. Mengurutkan Data

Untuk mengurutkan DataFrame:

                        df.sort_values(by='Umur', ascending=False)

11. Mengelompokkan Data

Untuk mengelompokkan data dan menghitung agregat:

                        df.groupby('Kota').count()

12. Menyimpan DataFrame ke File

Untuk menyimpan DataFrame ke file CSV:

                        df.to_csv('data.csv', index=False)

13. Kesimpulan

Pandas adalah alat yang sangat berguna untuk analisis data di Python. Dengan memahami dasar-dasar Pandas, kamu dapat melakukan manipulasi dan analisis data dengan lebih efisien.
