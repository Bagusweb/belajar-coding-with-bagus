Materi Linux

1. Pengenalan Linux

Linux adalah sistem operasi open-source yang digunakan di server, desktop, dan perangkat lainnya. Ia memiliki berbagai distribusi seperti Ubuntu, CentOS, dan Fedora.

2. Menginstal Linux

Untuk menginstal Linux, kamu perlu mendownload file ISO dari distribusi pilihanmu dan membuat bootable USB atau DVD. Kemudian, ikuti petunjuk instalasi di layar.

3. Struktur Sistem File Linux

Struktur sistem file Linux mencakup direktori utama seperti:

  • /: Root directory
  • /home: Direktori pengguna
  • /etc: File konfigurasi sistem
  • /var: File yang bervariasi, seperti log

4. Perintah Dasar Linux

Berikut adalah beberapa perintah dasar di terminal Linux:

  • ls: Menampilkan daftar file dan direktori.
  • cd: Mengubah direktori.
  • mkdir: Membuat direktori baru.
  • rm: Menghapus file atau direktori.

5. Manajemen Paket

Linux menggunakan manajer paket untuk menginstal dan mengelola perangkat lunak. Contoh manajer paket:

  • apt (Debian/Ubuntu): sudo apt install nama_paket
  • yum (CentOS): sudo yum install nama_paket

6. Pengguna dan Hak Akses

Linux menggunakan sistem pengguna dan grup untuk mengelola hak akses. Perintah untuk mengubah hak akses:

            chmod 755 nama_file

7. Shell dan Terminal

Shell adalah antarmuka pengguna untuk berinteraksi dengan sistem. Terminal adalah aplikasi yang digunakan untuk menjalankan shell.

8. Proses dan Manajemen Proses

Untuk melihat proses yang sedang berjalan, gunakan perintah:

            ps aux

Untuk menghentikan proses, gunakan:

            kill PID

9. Jaringan di Linux

Perintah jaringan yang umum digunakan:

  • ping: Menguji konektivitas ke host.
  • ifconfig: Menampilkan informasi konfigurasi jaringan.

10. Konfigurasi Firewall

Untuk mengelola firewall di Linux, kamu dapat menggunakan iptables:

            sudo iptables -L

11. Backup dan Restore

Untuk melakukan backup, kamu bisa menggunakan perintah tar:

            tar -cvf backup.tar /path/to/directory

12. Kesimpulan

Linux adalah sistem operasi yang kuat dan fleksibel. Dengan memahami perintah dasar dan konsep manajemen sistem, kamu dapat memanfaatkan potensi penuh dari Linux.
