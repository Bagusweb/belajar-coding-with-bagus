Materi GitHub

1. Pengenalan GitHub

GitHub adalah platform hosting untuk proyek yang menggunakan sistem kontrol versi Git. GitHub memudahkan kolaborasi antara pengembang dan memungkinkan manajemen proyek yang lebih baik.

2. Membuat Akun GitHub

Untuk menggunakan GitHub, kamu perlu membuat akun. Kunjungi GitHub.com, klik "Sign Up", dan ikuti langkah-langkah pendaftaran.

3. Membuat Repository

Repository adalah tempat penyimpanan untuk proyekmu. Untuk membuat repository baru:

  1. Masuk ke akun GitHub kamu.
  2. Klik tombol New di halaman repository.
  3. Isi nama repository dan deskripsi (opsional), lalu klik Create repository.

4. Mengkloning Repository

Untuk mengkloning repository ke komputer lokal, gunakan perintah:

            git clone [url-repository]

Ganti [url-repository] dengan URL repository yang ingin kamu kloning.

5. Menambahkan File ke Repository

Untuk menambahkan file baru ke repository:

  1. Gunakan perintah git add [file] untuk menambahkan file ke staging area.
  2. Setelah itu, lakukan commit dengan git commit -m "pesan commit".

6. Mengirim Perubahan ke GitHub

Setelah melakukan commit, kamu bisa mengirim perubahan ke GitHub dengan:

            git push origin main

Ganti main dengan nama branch yang kamu gunakan.

7. Mengambil Perubahan dari GitHub

Untuk mengambil dan menggabungkan perubahan terbaru dari GitHub ke repositori lokal:

            git pull origin main

8. Forking Repository

Forking adalah cara untuk membuat salinan repository di akun GitHub kamu. Ini berguna untuk berkontribusi pada proyek orang lain. Untuk melakukan fork, klik tombol Fork di pojok kanan atas halaman repository.

9. Pull Request

Pull request adalah permintaan untuk menggabungkan perubahan dari fork atau branch ke branch utama. Setelah melakukan perubahan, klik tombol New Pull Request di halaman repository untuk mengajukan permintaan.

10. GitHub Pages

GitHub Pages memungkinkan kamu untuk membuat situs web statis langsung dari repository. Untuk membuat GitHub Pages, pergi ke pengaturan repository, dan aktifkan opsi GitHub Pages.

Kesimpulan

GitHub adalah alat yang sangat berguna untuk kolaborasi dan manajemen proyek dalam pengembangan perangkat lunak. Dengan memahami dasar-dasar GitHub, kamu dapat berkontribusi pada proyek open-source dan mengelola proyek pribadi dengan lebih efektif.
