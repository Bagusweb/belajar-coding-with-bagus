Materi Pemrograman Go

1. Pengertian Go

Go (atau Golang) adalah bahasa pemrograman yang dikembangkan oleh Google. Go dirancang untuk menjadi sederhana, efisien, dan memiliki performa yang baik. Go mendukung pemrograman konkuren dengan goroutine yang memudahkan dalam pengelolaan thread.

2. Instalasi Go

Untuk menginstal Go, Anda dapat mengunjungi halaman resmi Go. Pilih versi yang sesuai dengan sistem operasi Anda dan ikuti instruksi instalasi.

3. Dasar-Dasar Go

a. Hello World

Program sederhana pertama yang ditulis dalam Go adalah "Hello, World!"

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

4. Tipe Data

Go memiliki beberapa tipe data bawaan, seperti:

  • int: Tipe data bilangan bulat.
  • float64: Tipe data untuk angka desimal.
  • string: Tipe data untuk teks.
  • bool: Tipe data untuk nilai true/false.

b. Contoh Tipe Data

package main

import "fmt"

func main() {
    var a int = 10
    var b float64 = 3.14
    var c string = "Hello"
    var d bool = true

    fmt.Println(a, b, c, d)
}

5. Struktur Kontrol

Go mendukung struktur kontrol seperti if, for, dan switch:

a. Contoh If

package main

import "fmt"

func main() {
    a := 10
    if a > 5 {
        fmt.Println("a lebih besar dari 5")
    } else {
        fmt.Println("a kurang dari atau sama dengan 5")
    }
}

b. Contoh For

package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}

6. Fungsi

Fungsi adalah blok kode yang dapat dipanggil untuk menjalankan tugas tertentu. Berikut adalah cara mendefinisikan dan memanggil fungsi:

package main

import "fmt"

// Mendefinisikan fungsi
func tambah(a int, b int) int {
    return a + b
}

func main() {
    result := tambah(5, 3)
    fmt.Println("Hasil:", result)
}

7. Struct

Struct adalah tipe data yang digunakan untuk mengelompokkan beberapa data menjadi satu unit:

package main

import "fmt"

// Mendefinisikan struct
type Mahasiswa struct {
    Nama  string
    Umur  int
}

func main() {
    // Membuat objek dari struct
    mhs := Mahasiswa{Nama: "Budi", Umur: 20}
    fmt.Println("Nama:", mhs.Nama, ", Umur:", mhs.Umur)
}

8. Goroutine dan Channel

Go mendukung pemrograman konkuren menggunakan goroutine dan channel:

a. Contoh Goroutine

package main

import (
    "fmt"
    "time"
)

func sayHello() {
    for i := 0; i < 5; i++ {
        fmt.Println("Hello")
        time.Sleep(100 * time.Millisecond)
    }
}

func main() {
    go sayHello() // Menjalankan fungsi dalam goroutine
    time.Sleep(1 * time.Second) // Menunggu goroutine selesai
}

b. Contoh Channel

package main

import (
    "fmt"
)

func main() {
    ch := make(chan string) // Membuat channel

    go func() {
        ch <- "Hello from goroutine"
    }()

    msg := <-ch // Menerima pesan dari channel
    fmt.Println(msg)
}

9. Error Handling

Go menggunakan nilai kembali untuk menangani error. Berikut adalah contoh bagaimana cara menangani error:

package main

import (
    "fmt"
    "strconv"
)

func main() {
    num, err := strconv.Atoi("123")
    if err != nil {
        fmt.Println("Terjadi error:", err)
    } else {
        fmt.Println("Angka:", num)
    }
}

10. Package Management

Go menggunakan Go Modules untuk manajemen package. Untuk menginisialisasi modul baru, gunakan perintah berikut:

go mod init nama_modul

Setelah itu, Anda dapat menambahkan dependensi dengan perintah:

go get nama_package

Kesimpulan

Pemrograman Go merupakan pilihan yang baik untuk pengembangan aplikasi modern. Dengan sintaksis yang sederhana dan dukungan untuk pemrograman konkuren, Go memungkinkan pengembang untuk membangun aplikasi yang efisien dan scalable. Fitur-fitur seperti tipe data, struktur kontrol, fungsi, struct, goroutine, dan manajemen error membuat Go sangat cocok untuk berbagai jenis aplikasi.

Pastikan untuk menjelajahi lebih lanjut tentang fitur-fitur lainnya seperti error handling, interface, dan package management untuk memperdalam pemahaman Anda tentang Go.
