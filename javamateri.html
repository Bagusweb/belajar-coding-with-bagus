Materi Java

1. Pengertian Java

Java adalah bahasa pemrograman berorientasi objek yang dikembangkan oleh Sun Microsystems (sekarang bagian dari Oracle). Java dirancang untuk memiliki sedikit ketergantungan pada platform, sehingga dapat dijalankan di berbagai perangkat dengan bantuan Java Virtual Machine (JVM).

2. Dasar-dasar Java

a. Sintaks Dasar

  • Pernyataan Cetak: Untuk mencetak teks atau variabel ke konsol. 
    System.out.println("Hello, World!");
  • Variabel: Harus dideklarasikan dengan tipe data. 
    String nama = "John";
int umur = 25;
  • Komentar: Gunakan `//` untuk komentar tunggal dan `/* ... */` untuk komentar multi-baris. 
    // Ini adalah komentar
/*
Ini adalah komentar multi-baris
*/

3. Tipe Data di Java

Java memiliki berbagai tipe data yang dapat digunakan untuk menyimpan berbagai jenis nilai.

  • String: Teks yang diapit oleh tanda kutip. 
    String teks = "Ini adalah string";
  • Integer: Bilangan bulat. 
    int umur = 25;
  • Double: Bilangan desimal. 
    double pi = 3.14;
  • Boolean: Menyimpan nilai True atau False. 
    boolean isValid = true;

4. Struktur Kendali

Java mendukung berbagai struktur kendali seperti perulangan dan percabangan.

  • IF Statement: Menggunakan `if`, `else if`, dan `else` untuk percabangan. 
    int umur = 20;
if (umur >= 18) {
    System.out.println("Dewasa");
} else {
    System.out.println("Belum Dewasa");
}
  • For Loop: Mengulang berdasarkan koleksi data. 
    String[] buah = {"apel", "jeruk", "pisang"};
for (String item : buah) {
    System.out.println(item);
}
  • While Loop: Mengulang selama kondisi benar. 
    int angka = 1;
while (angka < 5) {
    System.out.println(angka);
    angka++;
}

5. Fungsi di Java

Fungsi digunakan untuk mengelompokkan blok kode yang dapat dipanggil berulang kali.

  • Mendefinisikan Fungsi: Fungsi dibuat dengan tipe pengembalian dan nama fungsi. 
    void halo() {
    System.out.println("Halo Dunia!");
}
  • Fungsi dengan Parameter: Fungsi dapat menerima argumen. 
    void sapa(String nama) {
    System.out.println("Halo, " + nama + "!");
}
sapa("John");
  • Return Statement: Fungsi dapat mengembalikan nilai. 
    int tambah(int a, int b) {
    return a + b;
}
int hasil = tambah(3, 4);
System.out.println(hasil);

6. Pemrograman Berorientasi Objek (OOP)

Java mendukung pemrograman berorientasi objek (OOP) yang memungkinkan penggunaan kelas dan objek.

  • Membuat Kelas: Kelas adalah blueprint untuk membuat objek. 
    class Orang {
    String nama;
    int umur;

    Orang(String nama, int umur) {
        this.nama = nama;
        this.umur = umur;
    }

    void perkenalan() {
        System.out.println("Nama saya " + nama + " dan saya " + umur + " tahun.");
    }
}
  • Membuat Objek: Objek dibuat dari kelas. 
    Orang orang1 = new Orang("John", 30);
orang1.perkenalan();
  • Pewarisan: Kelas dapat mewarisi sifat dari kelas lain. 
    class Pelajar extends Orang {
    String sekolah;

    Pelajar(String nama, int umur, String sekolah) {
        super(nama, umur);
        this.sekolah = sekolah;
    }

    void perkenalan() {
        super.perkenalan();
        System.out.println("Saya sekolah di " + sekolah + ".");
    }
}

7. Antarmuka (Interface)

Antarmuka adalah kontrak yang dapat diimplementasikan oleh kelas untuk menyediakan metode tertentu.

interface Kendaraan {
    void bergerak();
}

class Mobil implements Kendaraan {
    public void bergerak() {
        System.out.println("Mobil bergerak.");
    }
}

8. Koleksi dan Struktur Data

Java menyediakan koleksi untuk menyimpan dan mengelola grup objek.

  • ArrayList: Struktur data dinamis yang dapat menyimpan elemen dalam urutan tertentu. 
    ArrayList daftar = new ArrayList<>();
daftar.add("apel");
daftar.add("jeruk");
  • HashMap: Struktur data untuk menyimpan pasangan kunci-nilai. 
    HashMap usia = new HashMap<>();
usia.put("John", 30);
usia.put("Jane", 25);

9. Penanganan Eksepsi

Java menggunakan blok `try-catch` untuk menangani eksepsi yang terjadi selama eksekusi program.

try {
    int hasil = 10 / 0; // Ini akan menimbulkan ArithmeticException
} catch (ArithmeticException e) {
    System.out.println("Terjadi kesalahan: " + e.getMessage());
}

10. Multithreading

Java mendukung multithreading, yang memungkinkan eksekusi beberapa thread secara bersamaan.

class ThreadContoh extends Thread {
    public void run() {
        System.out.println("Thread berjalan.");
    }
}
ThreadContoh thread = new ThreadContoh();
thread.start();

Kesimpulan

Java adalah bahasa pemrograman yang powerful dan memiliki komunitas yang besar. Dengan konsep OOP dan platform-independence, Java menjadi pilihan populer untuk pengembangan aplikasi desktop, web, dan mobile. Mempelajari Java membuka peluang karir yang luas dalam dunia teknologi informasi.
