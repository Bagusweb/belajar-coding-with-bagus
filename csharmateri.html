Materi C#

1. Pengenalan C#

C# adalah bahasa pemrograman modern yang dikembangkan oleh Microsoft. Bahasa ini berorientasi objek dan digunakan untuk pengembangan aplikasi di berbagai platform, termasuk desktop, web, dan mobile.

2. Menginstal C#

Untuk memulai dengan C#, kamu perlu menginstal Visual Studio. Pilih instalasi dengan .NET desktop development untuk mengembangkan aplikasi C#.

3. Struktur Dasar Program C#

Berikut adalah contoh struktur dasar program C#:

            using System;

            class Program {
              static void Main(string[] args) {
                Console.WriteLine("Hello, C#!");
              }
            }

4. Tipe Data di C#

C# memiliki beberapa tipe data dasar, seperti:

  • int: Untuk bilangan bulat.
  • double: Untuk bilangan desimal.
  • string: Untuk teks.
  • bool: Untuk nilai true/false.

5. Kontrol Alur (If, Switch)

C# menyediakan kontrol alur seperti if dan switch untuk mengambil keputusan:

            if (x > 10) {
              Console.WriteLine("x lebih besar dari 10");
            } else {
              Console.WriteLine("x tidak lebih besar dari 10");
            }

6. Looping (For, While)

Kamu dapat menggunakan loop untuk mengulangi blok kode:

            for (int i = 0; i < 5; i++) {
              Console.WriteLine(i);
            }

7. Fungsi dan Metode

C# memungkinkan kamu untuk mendefinisikan fungsi dan metode:

            static int Tambah(int a, int b) {
              return a + b;
            }

8. Kelas dan Objek

C# adalah bahasa berorientasi objek, yang berarti kamu dapat membuat kelas dan objek:

            class Mobil {
              public string Merek { get; set; }
              public void Nyala() {
                Console.WriteLine("Mobil menyala");
              }
            }

9. Warisan (Inheritance)

C# mendukung warisan, di mana kelas dapat mewarisi sifat dari kelas lain:

            class Kendaraan {
              public void Bergerak() {
                Console.WriteLine("Kendaraan bergerak");
              }
            }

            class Mobil : Kendaraan { }

10. Polimorfisme

Polimorfisme memungkinkan metode untuk memiliki perilaku berbeda tergantung pada objek yang memanggilnya:

            class Kendaraan {
              public virtual void Suara() {
                Console.WriteLine("Suara kendaraan");
              }
            }

            class Mobil : Kendaraan {
              public override void Suara() {
                Console.WriteLine("Mobil berbunyi beep");
              }
            }

11. Exception Handling

Untuk menangani kesalahan, C# menggunakan blok try dan catch:

            try {
              int x = 10 / 0;
            } catch (DivideByZeroException e) {
              Console.WriteLine("Tidak bisa membagi dengan nol");
            }

12. Kesimpulan

C# adalah bahasa pemrograman yang kuat dan serbaguna, cocok untuk berbagai jenis aplikasi. Dengan memahami konsep dasar yang telah dijelaskan, kamu dapat mulai mengembangkan aplikasi dengan C#.
