Materi Swift

1. Pengenalan Swift

Swift adalah bahasa pemrograman yang dikembangkan oleh Apple untuk membangun aplikasi di platform iOS, macOS, watchOS, dan tvOS. Swift dirancang untuk menjadi cepat, aman, dan mudah dipahami.

2. Menginstal Swift

Untuk mulai menggunakan Swift, kamu perlu menginstal Xcode di Mac. Setelah itu, kamu dapat membuat proyek baru dan memilih Swift sebagai bahasa pemrograman.

3. Struktur Program Swift

Berikut adalah struktur dasar dari program Swift:

            import Foundation

            print("Hello, World!")

4. Variabel dan Tipe Data

Swift mendukung beberapa tipe data, seperti:

  • Int: Bilangan bulat.
  • Double: Bilangan desimal.
  • String: Teks.
  • Bool: Nilai true atau false.

Contoh mendeklarasikan variabel:

            var nama: String = "John"
            let umur: Int = 30

5. Kontrol Alur

Swift mendukung struktur kontrol alur seperti if, switch, for, dan while. Berikut adalah contoh penggunaan if:

            let angka = 10
            if angka > 0 {
              print("Positif")
            } else {
              print("Negatif")
            }

6. Fungsi

Fungsi di Swift dideklarasikan dengan menggunakan kata kunci func. Berikut adalah contoh fungsi yang menerima parameter dan mengembalikan nilai:

            func tambah(a: Int, b: Int) -> Int {
              return a + b
            }

7. Kelas dan Objek

Swift mendukung pemrograman berorientasi objek. Berikut adalah contoh mendefinisikan kelas dan membuat objek:

            class Mobil {
              var merek: String
              var tahun: Int

              init(merek: String, tahun: Int) {
                self.merek = merek
                self.tahun = tahun
              }
            }

            let mobilSaya = Mobil(merek: "Toyota", tahun: 2021)
            print("\(mobilSaya.merek) \(mobilSaya.tahun)")

8. Pewarisan Kelas

Swift mendukung pewarisan kelas. Berikut adalah contoh kelas induk dan kelas turunan:

            class Kendaraan {
              var merek: String
              init(merek: String) {
                self.merek = merek
              }
            }

            class Mobil: Kendaraan {
              var tahun: Int
              init(merek: String, tahun: Int) {
                self.tahun = tahun
                super.init(merek: merek)
              }
            }

9. Koleksi

Swift menyediakan berbagai jenis koleksi seperti Array, Set, dan Dictionary. Berikut adalah contoh menggunakan Array:

            let daftarBuah = ["Apel", "Pisang", "Jeruk"]
            for buah in daftarBuah {
              print(buah)
            }

10. Penanganan Eksepsi

Swift mendukung penanganan eksepsi dengan menggunakan blok do-catch. Berikut adalah contoh:

            enum ErrorExample: Error {
              case divisionByZero
            }

            func bagi(_ a: Int, _ b: Int) throws -> Int {
              if b == 0 {
                throw ErrorExample.divisionByZero
              }
              return a / b
            }

              let hasil = try bagi(10, 0)

            }

11. Closures

Closures adalah blok kode yang dapat digunakan di dalam fungsi. Berikut adalah contoh penggunaan closures:

            let angka = [1, 2, 3, 4, 5]
            let kuadrat = angka.map { $0 * $0 }
            print(kuadrat)

12. Protokol

Protokol adalah kumpulan metode yang dapat diadopsi oleh kelas. Berikut adalah contoh:

            protocol Kendaraan {
              func suaraMesin()
            }

            class Mobil: Kendaraan {
              func suaraMesin() {
                print("Vroom Vroom")
              }
            }

Kesimpulan

Swift adalah bahasa yang kuat untuk pengembangan aplikasi di ekosistem Apple. Dengan memahami dasar-dasar Swift, kamu dapat mulai membuat aplikasi dengan lebih percaya diri.
